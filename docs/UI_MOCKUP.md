## Responsive Behavior

### Desktop (>1200px)
- Full 3-panel layout as shown
- Clothing preview: 200px max width
- All features visible

### Tablet (768px - 1200px)
- Left panel: 350px
- Right panel: 350px
- Clothing preview: 180px max width
- Slightly smaller buttons

### Mobile (<768px)
- Single column layout
- Panels stack vertically
- Clothing preview: Full width (max 300px)
- Larger touch targets for buttons

<button 
    id="transferClothingBtn"

### Keyboard Navigation
- All buttons focusable with Tab
- Enter/Space to activate buttons
- Escape to close preview (if modal)
- Visual focus indicators

### Screen Reader Support
- Status messages announced via `aria-live="polite"`
- Button states announced (enabled/disabled)
- Progress updates during transfer

## Animation Timing

### Panel Appearance
- **Duration:** 300ms
- **Easing:** `cubic-bezier(0.4, 0, 0.2, 1)`
- **Effect:** Fade in + slide down

### Preview Display
- **Duration:** 200ms
- **Easing:** `ease-out`
- **Effect:** Fade in + scale from 0.95 to 1

### Button State Changes
- **Duration:** 150ms
- **Easing:** `ease-in-out`
- **Effect:** Color transition

### Status Messages
- **Duration:** 250ms
- **Easing:** `ease-out`
- **Effect:** Slide in from top
